What is the Document Object Model (DOM)?

DOM allows programming languages to access, modify, and interact with the content, structure, and styles of a web page

How do you select an element from the DOM?

1. Selecting by ID: getElementById
2. Selecting by Class: getElementsByClassName
3. Selecting by Tag Name: getElementsByTagName
4. Selecting by CSS Selector: querySelector
5. Selecting Multiple Elements by CSS Selector: querySelectorAll

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in the DOM that allows to handle events at a higher level in the DOM tree and events that occur on the child elements are "delegated" to that parent through event bubbling.
